关于党吉圣教授

个人简介

党吉圣教授现任兰州大学副教授，硕士生导师。他于2020年获得兰州交通大学硕士学位，2025年获得中山大学博士学位，随后在兰州大学任教。

他的研究兴趣主要集中在人工智能、机器学习、大模型视频理解领域，在相关领域发表了几十篇学术论文，其中SCI收录10余篇以上，获得国家发明专利多项。

研究领域

党吉圣教授的研究集中在人工智能与视频理解交叉领域，致力于解决实际应用中的关键技术问题

人工智能与机器学习

专注于深度学习理论与应用，包括卷积神经网络、循环神经网络和生成对抗网络等方向的研究。

  • 高效神经网络架构设计
  • 小样本学习与迁移学习
  • 机器学习在信号处理中的应用

视频分割方法应用

研究先进视频分割方法框架在大模型框架中的应用，特别是理解速度和鲁棒性问题有独特的见解。

  • 同源记忆网络的高效视频分割
  • 时空并行稀疏记忆网络的高效视频分割
  • 逐帧和逐段时空交互记忆网络的高效视频分割

智能系统与应用

将人工智能与信号处理技术应用于实际系统，解决医疗、交通和工业等领域的问题。

  • 智能医疗诊断系统
  • 自动驾驶中的感知与决策
  • 工业物联网中的智能监测

论文发表

以下是党吉圣教授近年来发表的部分高质量学术论文，完整列表可在IEEE Xplore查看

Adaptive Sparse Memory Networks for Efficient and Robust Video Object Segmentation

党吉圣, Huicheng Zheng, Xiaohao Xu, Longguang Wang, Qingyong Hu, Yulan Guo
IEEE Transactions on Communications, vol. 68, no. 5, pp. 3256-3268, 2020

本文设计了一种自适应稀疏记忆构造器（ASMC），以根据视频帧中的动态时间变化自适应地记忆信息丰富的过去帧。此外，还引入了一种注意力局部记忆读取器（ALMR），利用一部分记忆快速检索相关信息，从而以更简便的方式减少帧级冗余计算和噪声。

期刊论文

Beyond Appearance: Multi-Frame Spatio-Temporal Context Memory Networks for Efficient and Robust Video Object Segmentation

党吉圣, Huicheng Zheng, Xiaohao Xu, Longguang Wang, Yulan Guo
Proceedings of the IEEE International Conference on Communications (ICC), pp. 1-6, 2021

引入了一种多帧时空上下文记忆（STCM）网络，通过利用多帧上下文交互模块（MCI）进行记忆构建，以挖掘多个相邻帧中有区分性的时空线索。基于所提出的 MCI 模块，开发了一种稀疏组记忆读取器，以在记忆读取过程中实现高效的稀疏匹配。

会议论文

Temporo-Spatial Parallel Sparse Memory Networks for Efficient Video Object Segmentation

Huicheng Zheng, Bimei Wang, Longguang Wang, Yulan Guo, 党吉圣
IEEE Journal of Biomedical and Health Informatics, vol. 25, no. 7, pp. 2893-2904, 2021

本研究提出了一种时空并行稀疏记忆网络（TSPSM），以实现高效的视频目标分割。我们的 TSPSM 并行构建了一个时间记忆库和一个空间记忆库，用于记忆互补的判别性目标线索。

Unified Spatio-Temporal Dynamic Routing for Efficient Video Object Segmentation
